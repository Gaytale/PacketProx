# Gaytale PacketProx

An earlyplugin to inject into the packet handler in order to log packets sent to and from the server.

# How to use:
- `git clone https://github.com/Gaytale/PacketProx`
- `cd PacketProx`
- Place a `HytaleServer.jar` in `libs` then import the project
- Build the project
- Place it in your server (or singleplayer world) in the `earlyplugins` alongside [Hyxin](https://github.com/Build-9/Hyxin)
