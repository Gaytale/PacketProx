# Gaytale PacketProx

A Hytale mod to intercept packets sent to and from the server in order to log them.

# How to use:
- `git clone https://github.com/Gaytale/PacketProx.git`
- `cd PacketProx`
- Place a `HytaleServer.jar` in `libs` then import the project
- Build the project
- Place the `/build/libs/Gaytale-PacketProx-VERSION.jar` file in your server (or singleplayer world) in the `mods` folder
